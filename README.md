# go-learning

go-language 学习（2021年寒假）

开发环境: `go version go1.20 linux/amd64`

* Docs: https://www.yuque.com/huangzhongqing/lang/xob788


## 运行
```shell
# 复杂
go build hello.go # 编译
./hello # 运行

# 常用
go run hello.go # 编译+运行一块

```



## Strength

* 开发效率，运行效率皆可
* 支持并发，最大的优势
* gc垃圾回收
* 适用于服务器编程，分布式系统，网络编程，内存数据库，云平台
